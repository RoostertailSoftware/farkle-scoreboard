Object:
The player with the highest score above {{config.winScore}} points on the final round of play wins!

Setup:
Start by adding Players using the `add Player` button.
Use the configuration dialog to set different variations of the game and scoring values.
Once all the Players are set, and game values are agreed on start the game with the `begin game` button

Game Play:
Farkle Scoreboard will promt each player in order. Player rolls 6 dice and chooses which to keep.
Select on the Scoreboard which dice are select to keep the points.

Score:
Only 1's and 5's are by themselves worth points.
  • 1's = 100 points
  • 5's = 50 points
rolling multiples of 3 or more of a die in a single roll are worth 100 * die value except 1's.
note: rolling 1's or 5's on three different rolls does not count as a triple. Farkel Scorecard will tally points correctly.

'Doubling' scoring on multiples is selected in configuration and points above are adjusted accordingly.

triple-doubles:
rolling any combination of 3 sets of doubles on a single roll is worth {{config.trippleDoublePoints}} points
example: 2-2-3-3-6-6
all 6 die are then used and the player can roll again, adding to his score if a valid roll.

double-triples:
rolling any combination of 2 sets of triples on a single roll is worth the "triples (3)" points from table above added
example: 2-2-2-6-6-6 would be 200 + 600 = 800 points
example: 1-1-1-4-4-4 would be 1000 + 400 = 1400 points
all 6 die are then used and the player can roll again, adding to his score if a valid roll.

straight:
rolling 1-2-3-4-5-6 on a single roll is worth {{config.straightPoints}} points
all 6 die are then used and the player can roll again, adding to his score if a valid roll.

Farkle:
if a player does not roll 1's, 5's or multiples described above on any roll during their turn the result is called Farkle!
Player looses all points accumulated on this turn only. Play passes to the next player.

If player rolls a Farkle {{config.maxFarkleCount}} times in consecutive turns, {{config.maxFarklePenality}} point will be deducted from the payer's score.
'maximum Farkles with penality' is selected in configuration
