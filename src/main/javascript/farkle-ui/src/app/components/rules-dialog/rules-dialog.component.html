Object:

The player with the highest score above {{config.winScore}} points on the final round of play wins!

Setup:

Start by adding Players using the `add Player` button.

Use the configuration dialogto set different variations of the game and scoring values.

Once all the Players are set, and game values are agreed on start the game with the `begin game` button

Game Play:

Farkle Scoreboard will promt each player in order. Player rolls 6 dice and chooses which to keep.

Select on the Scoreboard which dice are select to keep the points.

Score:

Only 1's and 5's are singularly worth points.

  • 1's = 100 points
  • 5's = 50 points

triples in any number are worth 100 points except 1's.

    Triples
  • 1's = 1000 points
  • 2's = 200 points
  • 3's = 300 points
  • 4's = 400 points
  • 5's = 500 points
  • 6's = 600 points
    Quad
  • 1's = 2000 points
  • 2's = 400 points
  • 3's = 600 points
  • 4's = 800 points
  • 5's = 1000 points
  • 6's = 1200 points
